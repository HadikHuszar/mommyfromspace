How I Was Able to Get My Code to Space

The Twilio Space Team Project at SIGNAL 2021

How It All Started

Have you ever wondered what messages people would send into space? As part of Twilio's Historic Mission to Send Code to space, they teamed up with the Talon Space Mission team to accomplish this ambitious, yet exciting, goal during Twilio's 2021 SIGNAL event.

TwilioQuest 3.2 launched at SIGNAL this year, and they wanted to do something big to celebrate. SIGNAL attendees had the once-in-a-lifetime opportunity to send some code to space aboard Talon 1, a balloon designed to send a camera-equipped Raspberry Pi into near Earth orbit.
